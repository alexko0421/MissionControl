通讯协议（JSON over Unix Socket）

Agent → App（上行）
// 状态更新
{ "type": "status_update",
  "agent": "claude-code",
  "status": "running",
  "task": "重构 API..." }

// Permission 请求
{ "type": "permission_request",
  "request_id": "abc123",
  "tool": "Bash",
  "command": "rm -rf dist/" }

// Plan 预览
{ "type": "plan_review",
  "request_id": "def456",
  "markdown": "## Plan\n..." }
App → Agent（下行）
// Permission 回应
{ "type": "permission_response",
  "request_id": "abc123",
  "decision": "approve" }

// Plan 回应
{ "type": "plan_response",
  "request_id": "def456",
  "decision": "approve" }

// 或者拒绝
{ "type": "permission_response",
  "request_id": "abc123",
  "decision": "deny" }

核心组件

MCSocketServer
App 内嵌嘅 Unix socket server，监听 agent 连接，管理所有 session
mc-bridge
轻量 socket client binary，hook 调用佢嚟同 app 通讯（取代直接写 status.json）
CLI (setup)
自动检测已安装嘅 agent，注入对应 hook config，部署 mc-bridge
